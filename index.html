Item Description & Personalisation Instructions

Personalised Kraft Gift Tags with Optional Eyelet and Twine

This listing is for a set of Kraft brown gift tags with your own text printed on them in the font provided as per the listing images. These are ideal for all occasions and make any gift or item that little bit more special with your own personalised text. You can choose the pack size you require along with optional Silver eyelet or twine

Please read below for detals on sample tags and how to provide the text you want printed. Please read this carefully to ensure we can provide your tags with no unecessary delay.

Availble Pack sizes are - 1 Sample tag (see details below on sample tags), Single Personalised Tag, Set of 9, 15, 18, 24 or 27. For larger orders where Twine is selected the Twine will be included but will not be looped through the tags.

Specifications:

  • Colour: Kraft Brown (only 1 sided. The back is a generic grey card)
  • Length: 71mm
  • Width: 51mm
  • Thickness: 280gsm card
  • Optional Extras: 4mm Silver Eyelet and/or Length of Twine (approx. 260mm/26cm, *13cm when folded*)

SAMPLE TAGS

Sample tags will be provided with only sample text, and will NOT include the silver eyelet or twine, as these are only for the purpose of judging the size and look before making a larger order.

Please do not ask for custom text to be printed on a sample tag, as this will not happen and you will receive the sample with 'Your text here' only printed on them.

If you would like a single tag with personalised text printed on it please choose the option for 'Single Personalised Tag' and follow the personalisation instructions below.

I can produce a mockup for any larger orders. Please contact me prior to placing the order or alternatively state this in the message to seller so I can ensure this is provided before producing your tags.

PERSONALISATION INSTRUCTIONS

(PLEASE READ CAREFULLY)

  • Please supply the text you require printed on your tag via the message to seller when placing your order. This will allow me to match your order with the text you require. Failing to do this may result in a delay to your order. If you do not require any text and would like the tags left empty please state this in the message to seller as this can also cause a delay if not specified.

  • If you require a list of names printed please provide these in the same manner as above, seperating each name with a comma. If a name is required more than once please list them multiple times to make sure this is clear. If you have ordered the closest number of tags, but do not have enough names/text I will provide the remaining tags blank with no text printed, so they can be used as generic tags.

  • If you require a message printed please try keep this as short as possible, otherwise the font may look too small on the tag. I have no limitation on characters, but do ask you to keep this in mind. The size of the tag is detailed in the listing to help with this.
